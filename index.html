Generador de JSON aleatorios

Herramienta para generar JSON de prueba en forma sencilla.

Para generar su JSON seleccione uno o varios de los sets de valores precargados en el orden que quiere que aparezcan.
O añada sus propios sets personalizados utilizando Llave y Valores para cada campo.
Use comas (,) para separar los valores utilizando comillas para cada valor, si correspondiera.
Cuando este listo seleccione la cantidad de entradas del JSON y genere el archivo.
