Random JSON Generator

Simple tool to easy generate sample JSON files.

To generate your JSON file select one or many of our samples in the order you like them to appear.
Or Add Key and sample values on each fieldset. Add as many fieldsets as you want.
Use comma (,) to separaye values. And insert values in JS style.
When you´re done select the sample size and generate the sample file.
